**前端** ：HTML5、CSS3、Sass、ES6(7|8)、Webpack4、Vue CLI 3、Vue.JS2.6.10、Vuex3.0.1、Vue Router3.1.3、Axios0.19.0   

**UI框架选型** ：View UI 

**后端** ：Node.js（express ）、MySQL
